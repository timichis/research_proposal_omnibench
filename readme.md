# Анализ статьи

**Автор:** Бовкун Тимофей Дмитриевич

Данный репозиторий содержит материалы исследования и код эксперимента, проведенного в рамках анализа статьи *«What Limits Virtual Agent Application? OmniBench: A Scalable Multi-Dimensional Benchmark for Essential Virtual Agent Capabilities»*. 

## Описание

В анализируемой статье предложен графовый бенчмарк **OmniBench**, где текстовые инструкции представляются в виде DAG атомарных подзадач.

В рамках данного исследования проанализирован метод, описанный в статье, выделены ключевые задачи, оценены сильные и слабые стороны работы, предложены пути развитие статьи с целью её улучшения, а также проведен эксперимент, проверяющий следующую **гипотезу**: 
> Использование одной модели для генерации и валидации текстовой инструкции вносит алгоритмическую предвзятость. Сгенерированные данные будут соответствовать паттернам рассуждения этой же сети, что может исказить объективность (семантический резонанс).

## Структура репозитория

* **`main.pdf`** — Research proposal
* **`experiment.ipynb`** — Jupyter Notebook с кодом эксперимента. 
* **`report.md`** — Отчет с результатами эксперимента.
